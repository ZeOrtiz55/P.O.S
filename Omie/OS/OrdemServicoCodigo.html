viaUnica

cUtilizaViaUnica string1
Indica se o serviço utiliza Nota Fiscal Via Única (Modelos 21 e 22).
+
Pode ser:
S - Sim
N - Não
cModeloNF string2
Indica qual modelo de NF será emitida para o serviço.
+
Obrigatório quando a tag cUtilizaViaUnica for igual a "S".

Pode ser:
21 - Nota Fiscal de Serviço de Comunicação
22 - Nota Fiscal de Serviço de Telecomunicação
cCFOP string10
Código CFOP para NF Via Única (Modelos 21 e 22).
+
Obrigatório quando a tag cUtilizaViaUnica for igual a "S".

Utilize a tag "nCodigo" do método "ListarCFOP" da API /api/v1/produtos/cfop/ para obter essa informação.
cClassifServico string4
Código da Classificação para NF Via Única (Modelos 21 e 22).
+
Obrigatório quando a tag cUtilizaViaUnica for igual a "S".

Utilize a tag "cCodigo" do método "ListarClassificacaoServico" da API
/api/v1/servicos/classificacaoservico/ para obter essa informação.
cTipoReceita string1
Indicador do Tipo de Receita para NF Via Única (Modelos 21 e 22).
+
Obrigatório quando a tag cUtilizaViaUnica for igual a "S".

Pode ser:
0 - Receita própria - serviços prestados
1 - Receita própria - cobrança de débitos
2 - Receita própria - venda de mercadorias
3 - Receita própria - venda de serviço pré-pago
4 - Outras receitas próprias
5 - Receitas de terceiros (co-faturamento)
9 - Outras receitas de terceiros
cTipoUtilizacao string1
Código do tipo de utilização para NF Via Única (Modelos 21 e 22).
+
Obrigatório quando a tag cUtilizaViaUnica for igual a "S".

Utilize a tag "cCodigo" do método "ListarTipoUtilizacao" da API
/api/v1/servicos/tipoutilizacao/ para obter essa informação.
